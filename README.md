A blockchain-based grading system that allows academic results to be securely recorded, stored, and verified using smart contracts. This removes the need for a central authority and ensures transparency, integrity, and trust in the grading process.
This project is a decentralised application (DApp) designed to manage student grades on the blockchain. Instead of storing results in traditional databases, grades are written to a smart contract, making them tamper-proof and permanently recorded.

The system allows authorised users (e.g. proffessors) to assign grades, while students can view and verify their results directly.
