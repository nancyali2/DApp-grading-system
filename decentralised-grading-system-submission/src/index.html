addInstructor(address[])

Comma-separated list of wallet addresses to register as instructors.

getInstructorsList()

Returns all Ethereum addresses currently registered as instructors.

addCourse(courseID, name, rolls, addrs, TAs)

Short unique identifier, e.g. EE465 or CS101.
Comma-separated roll numbers in the same order as addresses below.
One address per student, matching the roll number order above.
Leave blank if no TAs. TAs can add/update marks but not calculate grades.

addExam(courseID, examID, maxMarks, rolls, marks)

e.g. quiz1, midsem, endsem. Must be unique within the course.
Comma-separated marks in the same order as roll numbers above.

updateMarks(courseID, examID, rolls, marks)

Only supply roll numbers where corrections are needed.

calculateGrades(courseID, weightages, cutoffs)

One value per exam in the order they were added. Must sum to 100.
Seven cutoffs for: AA, AB, BB, BC, CC, CD, DD. Must be strictly descending.
